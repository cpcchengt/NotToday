# NotToday

NotToday is a desktop-first floating event manager for Windows.

The goal is to make recording and managing dated events quick and distraction-free.

## Current Features

- Desktop application
- Add local events with a title and date
- View events in chronological order
- Mark events as completed
- Edit and delete events
- Floating window
- SQLite storage

## Planned Features

- Calendar
- Reminder
- Habit tracking
- Statistics
- Cloud sync
- AI assistant

## Stack

- Tauri
- React
- TypeScript
- SQLite

## Build

Build the frontend:

```bash
pnpm build
```

Package the desktop app:

```bash
pnpm tauri build
```
